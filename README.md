# Customer-Segmentation-Analysis
Advanced behavioural analytics on 50,000+ global consumer records. Utilised RFM modelling, PCA for dimensionality reduction, and K-Means clustering to derive actionable business segments.

## 🚀 Project Impact
Successfully engineered a scalable segmentation engine for **50,500+ global consumers**[cite: 1]. By implementing **Unsupervised Machine Learning**, I identified 5 distinct high-value personas, providing a roadmap for targeted marketing and retention strategies[cite: 2].

## 🛠 Advanced Analytics Stack
* **Big Data Handling**: Processed and cleaned a 50k+ row dataset with automated statistical outlier detection[cite: 1].
* **Dimensionality Reduction**: Utilized **PCA** to optimize clustering performance and visualize complex consumer behavior[cite: 2].
* **Clustering Logic**: Implemented **K-Means++** with centroid optimization[cite: 2].
* **RFM Engine**: Developed a proprietary scoring system to analyze Recency, Frequency, and Monetary value[cite: 2].
